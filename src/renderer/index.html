How to use

You can paste any Image File, Giphy or Video URL like Pexels, Instagram, Twitter, Youtube or any .mp4 file.

Record Audio: Hit R to start record audio.

Record video: Select the video source with Alt+Shift+Control+A and use Alt+Shift+Control+S to the record video

Then when image, audio or video appears, you can drag and drop it in any program that accepts dropping. OR after dragging, you'll find the file in the veditbox folder in your operational system user's folder
